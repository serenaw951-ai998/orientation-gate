### [music-prompt-engine-zh-v3.1.html](https://serenaw951-ai998.github.io/orientation-gate/tools/music/music-prompt-engine-zh-v3.1.html)
**v3.1 · 完整版（中文）** ✅ 最新
- 包含 v2 所有功能
- 新增情绪弧线预设（平稳流动 / 缓慢上升 / 积累与释放 / 波浪起伏）
- 输出分两栏：**风格标签**（粘贴到 Suno 风格栏）+ **情绪与结构**（粘贴到描述栏）
- 每栏独立 Copy 按钮 + 字符数显示
- 滑块新增说明文字（如：暗 = 小调 · 亮 = 大调）
- 滑块数值文字 + 数字同步显示（如：强烈 · 8）
- 情绪关键词一键预设（希望+坚定 / 心碎 / 思念 / 黑暗抗争）
- 高级设置折叠面板：BPM / 调性 / 节奏型
